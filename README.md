# 🌿 Digital Home (React Version)

A multi-page portfolio built with **Next.js and React**, designed as a cozy, user-centered digital space.

This project was created as part of my React coursework, with a focus on **component-based architecture, responsive design, and thoughtful user experience**.

---

## ✨ Features

- 🏡 Multi-page layout (Home, About, Projects, Contact)
- 🧩 Reusable components (Project Cards, Layout)
- 📱 Fully responsive design (mobile + desktop)
- 🎨 Custom styling with Bootstrap + CSS
- 📁 Dynamic project rendering using structured data
- 📄 Resume download functionality
- 🌙 I used this assignment to test out the design I created to update my existing portfolio

---

## 🛠️ Tech Stack

- React
- Next.js
- TypeScript
- Bootstrap
- CSS

---

## 🌸 About This Project

This portfolio reflects my approach to development:

> **Usability builds the house.  
> Emotional design makes it a home.  
> Logic keeps it safe.**

I focused on creating a space that feels welcoming and intuitive, while also demonstrating clean structure and practical functionality.
